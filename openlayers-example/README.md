## openlayers6-图形绘制
<img src="https://images.gitee.com/uploads/images/2020/0619/160359_dffc2d6e_5074164.png"  height="600" width="1000">

### 基于案例：[plot4ol3](https://gitee.com/ilocation/plot) https://gitee.com/ilocation/plot
### 原有功能包括：
- 标绘符号绘制；
- 标绘符号编辑；
- 实现的线状符号：弧线、曲线、折线、自由线；
- 实现的面状符号：圆、椭圆、弓形、扇形、曲线面、集结地、多边形、自由面；
- 实现的箭头符号：钳击、直箭头、细直箭头、突击方向、进攻方向、进攻方向（燕尾）、分队战斗行动、分队战斗行动（燕尾）；
### 新增功能包括：
- 可以绘制文字、矩形、五角星、正N边形；
- 支持在图形上面右键操作，进行删除、克隆操作，如果绘制的是文字类型，右键可以修改文字内容；
- 可以设置需要绘制图形的背景色、边框色；
- 显示关键点数据(地图上所有图形的关键点坐标信息)；
- 保存到缓存(暂时保存到cookie里面的，重新刷新页面可以看到保存之前的效果)；
- 清空地图(清空当前地图上面绘制的所有图形)；
- 删除最后操作图形，类似Ctrl+Z撤销操作；
- 导出图片(把当前地图可以显示的区域保存为一张图片)；
